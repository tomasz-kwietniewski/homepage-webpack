Specyficzność w CSS

Jak tabela medalowa. Najpierw sortujemy po najważniejszym medalu. W przypadku remisu patrzymy na mniej ważny medal itd.
Kolejność od najważniejszego (najwyższa specyficzność): !important, style="", #name, .name, element.

Dodawać klasy do wszystkich elementów i unikać zagnieżdżeń.

Nazewnictwo klas

Używaj pojedynczych słów z języka angielskiego; nazywaj rzeczy zgodnie z ich funkcją na stronie; startuj z małej litery np. header a nie Header; nie zaczynaj od cyfry; jeśli musisz używać dwóch słów, użyj tzw. kebab case np. news-listing; staraj się, żeby element miał jedną klasę (nie zwiększaj specyficzności bez potrzeby); tworzymy klasy nawet kiedy mamy jeden element danego typu.
